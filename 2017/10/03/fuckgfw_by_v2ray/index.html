    • v2ray 科学上网，Shadowsocks 之外的另外一个选择

    得益于 GFW 的不断严格的封锁和精准的流量拦截，之前长期稳定使用的 SS 已经挂掉，不得不寻求其他的解决方案，经过搜索。发现了使用了 Vmess 协议的 v2ray，决定尝试一下。官方的用户手册 写的非常详细，按照步骤操作，很容易搭建，我在这个过程中也遇到了一些其他问题，记录下来，且当备忘。

    准备

    • 一台能够直连外网 VPS，毫无疑问，如果你打算自己折腾，这是必须的。我使用的 Linode，系统版本是 Ubuntu 16.04
    • Mac 和 iOS 设备
    • uuid 在线生成工具：https://www.uuidgenerator.net/ ，该 uuid 后期将作为配置里的密码使用
    • Mac 和 PC 上的安装包
    • 美区 Apple ID
    • 支持 Visa 的信用卡 (非必需)

    服务端安装运行

    安装及其简单，只需要执行一行命令：

    bash <(curl -L -s https://install.direct/go.sh)

    自动安装成功之后，你会看到 v2Ray v.x.x.x is installed 的提示。主程序和配置文件，分别在

    编辑配置文件 /etc/v2ray/config.json，进行服务器配置。

    记下上面配置中的 port、id、alterId 三项，客户端配置的时候需要用到。

    服务端运行 v2ray，执行 service v2ray start 启动 v2ray。后续可以使用 service v2ray (start|stop|status|reload|restart|force-reload) 命令来控制，至此服务端的已经配置好并且开始运行。

    客户端配置

    iOS 配置

    iOS 上 shadowrocket 和 shadowray 是同一作者开发，均支持 vmess 协议。这两个 app 都已经在中区下架，需要注册美区帐号区购买。至于如何注册美区帐号，可以自行搜索，教程很多。还有一点需要注意的是这两款 app 是需要购买的，而我无论使用招行支持 visa 的全币种卡，还是 paypal 都无法购买，你也很有可能遇到这个情况。解决的方案，使用你的美区帐号登录苹果网站，用支持 visa 的卡购买 iTunes Gift Card 礼品卡充值，如果没有信用卡，还可以求助万能的淘宝(注意风险)。从购买到收到大概需要几个小时，激活之后就可以在 app store 使用礼品卡购买上面两个 app 了。

    Mac 上配置

    在下载列表里下载 Mac 上的安装包 v2ray-macos.zip，解压安装之后，打开 Configure… 按照提示配置即可。

    至此，V2ray 配置全部完成，可以愉快的使用新梯子爬过长城，使用谷歌高质量的搜索，观看 Youtube 上创意无穷的视频，或是刷 Twitter 了。
