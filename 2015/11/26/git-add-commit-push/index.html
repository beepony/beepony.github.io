这样就既节省了输入量，又能灵活的提交 commit 的信息，看起来似乎完美解决了我的问题。但又跟 git 设计之初 workflow 的初衷相违背，不过凡事处处满意也很难吧。算是做个简单的记录，满足了懒人的需求罢了，不必纠结。
